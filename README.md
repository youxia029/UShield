## 使用说明

1. 双击 UShield.exe 启动程序
2. 使用 admin / admin123 登录
3. 点击「启动监控」开始监听U盘
4. 插入U盘自动拍照记录

- **最小化**: 自动缩至系统托盘（任务栏右侧）
- **退出**: 点击退出需验证管理员密码
- **系统要求**: Windows 10/11 64位

## 核心功能

- USB 插拔实时监控，自动拍照取证
- 设备白名单/黑名单策略管控
- 文件操作审计（新建、修改、删除、重命名）
- 截图审计与回查
- 仪表盘数据统计
- 可选自动禁用未授权设备
